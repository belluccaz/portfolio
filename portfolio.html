A Simulação (os projetos):

Projeto 1

Este é um jogo da memória baseado no clássico Simon Game. Ele utiliza JavaScript para criar uma experiência interativa onde os usuários devem repetir sequências de cores geradas aleatoriamente. Conforme o jogo avança, a sequência se torna mais longa e difícil, desafiando a memória e a atenção dos jogadores.

Projeto 2: Análise de dados e deploy simples (Python)

Nesse aplicativo, você pode inserir diversos números para realizar análises estatísticas básicas. Desenvolvido em Python e utilizando a biblioteca Streamlit, este projeto visa simplificar o processo de análise de dados para usuários que não possuem experiência técnica avançada. Ideal para apresentações rápidas e insights imediatos.

Projeto 3: Drum Kit

Projeto simples de um "Drum Kit" virtual, onde você pode tocar bateria utilizando o teclado do computador. Cada tecla corresponde a um som de instrumento diferente, permitindo que os usuários criem suas próprias batidas e ritmos. Este projeto foi construído com HTML, CSS e JavaScript.

Projeto 4: EBAC Shoes

Projeto de website para uma loja fictícia de calçados. Desenvolvido utilizando HTML, CSS e JavaScript, este site possui um layout moderno e responsivo, com seções dedicadas para exibir produtos, promoções e informações de contato. O objetivo deste projeto é demonstrar habilidades de design front-end e usabilidade.

Projeto 5: Base Decrypter

O Conversor de Bases é uma aplicação web interativa que permite aos usuários converter números entre diferentes sistemas de numeração: binário, octal, decimal e hexadecimal. O projeto utiliza jQuery para manipulação do DOM e efeitos visuais, incluindo um plano de fundo dinâmico inspirado no efeito "Matrix".
